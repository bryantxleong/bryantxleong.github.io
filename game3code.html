Game Mechanic Code

As shown in the code sequence above, the initial conditions of game are set:

The score starts at 0 and the player has 3 lives. After that, the game begins with Stage 1.

In both Stage 1 and Stage 2, the exit doors are closed by default. The player must find a way to clear the stage, which will then open the doors. Once the player touches the door, they can proceed to the next stage.

Stage 3 is the Boss battle. When the player defeats the Boss, the game shows a screen congratulating the player for completing the game.
